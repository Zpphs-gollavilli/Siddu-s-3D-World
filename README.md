✨ **Siddu’s 3D World** is my interactive portfolio built with **React + Three.js + TailwindCSS**, where creativity meets 3D animations. Inspired by modern portfolio designs, but reimagined in my own style.  

---

## 🚀 Features & Animations
- 🌌 Immersive 3D world built with **Three.js**
- 🎭 Smooth transitions using **Framer Motion**
- 🌀 Animated models (customized character + elements)
- 📱 Fully responsive across devices
- 🎨 Minimal & modern UI with **TailwindCSS**
- 🖼️ Showcases my projects, skills, and creative work

## 🛠️ Tech Stack  
- ⚛️ **React + Vite** – frontend framework  
- 🎨 **Tailwind CSS** – modern styling  
- 🌐 **Three.js & React-Three-Fiber** – 3D animations  
- ✨ **Framer Motion** – smooth page transitions
